# spotify-playlists
app for searching through playlists in your country
# Showcase:
 https://youtu.be/txeFkeDI8X0

# Display link:
https://exp.host/@barakao/spotifyplaylists

# To run dev mode:

`git clone`
`cd repo`
`yarn`
`cd ios`
`pod install`
`cd ../`
`yarn test`
`expo start`

Don't forget to add your configurations to spotify app
